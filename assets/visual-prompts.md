# Librería de Prompts Visuales ARVION (calidad + variedad)

> El robot DEBE usar estos prompts (adaptando el sujeto al tema). Objetivo: imágenes **cinematográficas, ricas y VARIADAS** — nunca planas, nunca rejillas tenues con puntitos, nunca fondos vacíos.

## ⛔ REGLAS DE CALIDAD (obligatorias en CADA imagen)
- SIEMPRE un **sujeto focal brillante y nítido** (núcleo, hologramas, ciudad, partículas…), NO un fondo vacío.
- **Luz volumétrica + brillo (bloom) + alto contraste + 4K, fotorrealista, cinematográfico.**
- Paleta base: azul eléctrico ARVION + navy profundo (algunas pueden sumar acentos cálidos/dorados para variar).
- **PROHIBIDO:** fondos planos, rejillas tenues con puntitos, siluetas oscuras borrosas, espacios vacíos, looks genéricos.
- Siempre `no people, no text, no letters` y dejar **espacio inferior** para el texto.
- **VARÍA la escena en CADA pieza** — no repitas el mismo encuadre. Recorre la lista.
